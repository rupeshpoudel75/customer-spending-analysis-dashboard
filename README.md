# 📊 Customer Spending Analysis Dashboard

## 📌 Project Overview
This project analyses customer transaction data to identify spending patterns, sales trends, and top-performing products.

## 🎯 Objectives
- Analyse monthly sales trends
- Identify top customers
- Compare sales by gender and product category

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Bar, Pie, Column)
- Slicers

## 📈 Key Insights
- Electronics generated the highest revenue
- Female customers contributed more to total sales
- Sales increased significantly in April
